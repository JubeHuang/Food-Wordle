### 程式應用
* enum
* struct
* UIView.transition 翻字母卡牌動畫
* DispatchQueue.main.asyncAfter 延遲動畫
* UIAlertController Alert對話視窗
---

### 遊戲規則
* 猜出五個英文字母組成的食物相關單字
* 共有八次猜題機會
* 字母變綠色表示字母與位置皆正確、字母變黃色表示字母正確但位置不正確、字母變深灰色表示字母與位置皆錯誤
